* Prerequisites

- [[https://clojure.org/guides/getting_started][Clojure command-line tools]]
- [[https://babashka.org/][Babashka]] (optional)

* Running

To run with Clojure (JVM):

#+begin_src shell
  clojure -X:run
#+end_src

To run with Babashka:

#+begin_src shell
  bb src/twelve_days_of_clojure/song.clj
#+end_src

** REPL

To start a REPL:

#+begin_src shell
  clj -A:dev
#+end_src

At the REPL, run:

#+begin_src clojure
  (require '[twelve-days-of-clojure.test-harness :refer [run]])
  (run)
#+end_src

* My Rules

My totally made-up, self-imposed rules for this challenge:

1. Written in [[https://clojure.org/][Clojure]]
2. Must work on JVM Clojure, [[https://clojurescript.org/][ClojureScript]], and [[https://babashka.org/][Babashka]]
3. No libraries beyond those bundled with Clojure
4. Define only one macro

I have also decided that the the literal text of the program
must include the [[https://www.lyricsforchristmas.com/christmas-carols/the-twelve-days-of-christmas/][lyrics of the song]] as Clojure symbols,
without the repetition of past verses, like so:
